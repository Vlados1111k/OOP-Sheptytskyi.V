# Лабораторна робота №5

Цей проект — консольна система бронювання місць у кінотеатрі. 

### Особливості
1.  **Generic Matrix<T>**: Власний контейнер для зберігання схеми залу.
2.  **Композиція**: Зал (`CinemaHall`) складається з місць (`Seat`).
3.  **Винятки**: Кастомний `SeatAlreadyBookedException` для обробки колізій.
4.  **LINQ**: Використовується для підрахунку доходу та завантаженості залу.
