# Лабораторна робота №6: Делегати, Лямбда-вирази та LINQ

## Завдання
Розробити консольний додаток для демонстрації роботи з делегатами, анонімними методами, лямбда-виразами та бібліотекою LINQ.

## Що реалізовано:
1. **Власний делегат**:
   - Створено делегат `MathOperation` для виконання арифметичних дій.
2. **Лямбда-вирази та Анонімні методи**:
   - Показано різницю між старим підходом (delegate) та сучасним (lambda).
3. **Вбудовані делегати**:
   - `Func<double, bool>` — для перевірки умов (логіка).
   - `Action<double>` — для виведення результатів (UI).
   - `Predicate<int>` — для перевірки чисел.
4. **Робота з колекціями (LINQ)**:
   - Використано методи розширення: `Where` (фільтрація), `OrderBy` (сортування), `Select` (проєкція), `Aggregate` (акумуляція результату).
5. **Практичний приклад**:
   - Клас `TemperatureRecord` для аналізу погоди за тиждень.

## Висновок
Під час виконання лабораторної роботи я навчився використовувати делегати для створення гнучкого коду. 
Зрозумів, що лямбда-вирази значно скорочують написання коду порівняно з анонімними методами. 
Також опанував основи LINQ, що дозволяє обробляти масиви та списки в один рядок без громіздких циклів `foreach`.
