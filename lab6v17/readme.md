Звіт до Лабораторної роботи №6
Тема: Делегати, Лямбда-вирази та LINQ

Хід роботи:

Делегати: Реалізував логіку через вбудовані Func та Action, щоб не створювати зайві типи даних.

Лямбди: Замінив стандартні методи на лямбда-вирази (=>), що скоротило код у кілька разів.

LINQ: Використав ланцюжок методів (Where, OrderBy, Select) для обробки списку даних. Це дозволило відмовитися від громіздких циклів foreach.

Висновок: Навчився писати компактний код. Зрозумів, що використання LINQ та лямбда-виразів дозволяє вирішувати складні задачі з обробки даних (фільтрація, сортування) в один рядок.
