# Звіт до IndependentWork11

Виконане завдання демонструє механізми відмовостійкості (Resiliency) за допомогою бібліотеки **Polly**.

### Реалізовані сценарії:
1. **Retry (Повтор):** Автоматично робить 3 повторні спроби при збої API. Вирішує проблему короткочасних помилок мережі.
2. **Fallback (Запасний план):** Повертає дефолтне значення, якщо основний сервіс недоступний. Запобігає падінню застосунку.
3. **Circuit Breaker (Запобіжник):** Тимчасово блокує запити після серії помилок. Захищає зовнішню систему від перевантаження ("Fail Fast").
